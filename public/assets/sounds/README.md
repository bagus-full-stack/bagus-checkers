# Audio Assets

This folder contains sound effects for the Angular Checkers Master game.

## Required Sound Files

Create or add the following audio files (MP3 format recommended):

| File | Description | Suggested Duration |
|------|-------------|-------------------|
| `move.mp3` | Piece movement sound | 0.2-0.3s |
| `capture.mp3` | Piece capture sound | 0.3-0.5s |
| `promotion.mp3` | Pawn promotion to King | 0.5-0.8s |
| `check.mp3` | Check/danger sound | 0.3-0.5s |
| `game-start.mp3` | Game start jingle | 0.5-1.0s |
| `game-end.mp3` | Game end sound | 0.8-1.5s |
| `click.mp3` | UI click sound | 0.1-0.2s |
| `error.mp3` | Error/invalid action | 0.2-0.3s |
| `notification.mp3` | Notification sound | 0.3-0.5s |
| `tick.mp3` | Timer tick (soft) | 0.1s |
| `low-time.mp3` | Low time warning | 0.3-0.5s |
| `chat.mp3` | Chat message received | 0.2-0.3s |
| `background-music.mp3` | Background music (optional) | Loop |

## Free Resources

You can find free sound effects at:
- [Freesound.org](https://freesound.org)
- [Mixkit](https://mixkit.co/free-sound-effects/)
- [Pixabay](https://pixabay.com/sound-effects/)
- [Zapsplat](https://www.zapsplat.com)

## Tips

- Keep file sizes small (< 50KB each for effects)
- Use MP3 format for best browser compatibility
- Normalize audio levels for consistency
- For the tick sound, make it subtle to avoid annoyance
